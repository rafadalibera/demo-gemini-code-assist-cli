# Project Overview

This project is a simple web application that converts a number to its Roman numeral representation. It is built using Python and the Flask framework. The application has a main page with a form to enter a number and a results page that displays the converted Roman numeral.

# Building and Running

To run this project, you need to have Python and Flask installed.

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the application:**
    ```bash
    python main.py
    ```

    The application will be available at `http://0.0.0.0:8080`.

# Development Conventions

*   The main application logic is in `main.py`.
*   The conversion logic is in `converter.py`.
*   HTML templates are located in the `templates` directory.
*   The project follows standard Python coding conventions.
